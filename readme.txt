○概要
  少し面倒な文字列の処理を行うことを目的としたWebサイトです。
  Trim    …  前後のスペースを取り除きます。
  Concat  …  行ごとに文字列の結合を行います。
  Insert  …  すべての行に指定の文字を付与します。
